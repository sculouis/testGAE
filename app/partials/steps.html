Django建置步驟

  1. 建立一個網站根目錄．例如Website
  2. 進入網站根目錄，cd Website;django-admin.py startupproject mysite
  3. cd mysite;django-admin.py startapp MyBlog
  4. 進入WebSite目錄然後下tree可看到網站的目錄結構
  5. 進入網站應用程式（MyBlog）更改models.py
  6. python manage.py syncdb
  7. python manage.py runserver 127.0.0.1:8080
